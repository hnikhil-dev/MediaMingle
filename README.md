# MediaMingle

Your personalized entertainment hub for movies, TV shows, and anime.

## Features
- Search and browse movies, TV shows, and anime
- Mood-based recommendations
- Genre filtering
- Trending content

## Tech Stack
- Frontend: React
- Backend: FastAPI (Python)
- APIs: TMDb, Jikan (MyAnimeList)

## Deployment
- Backend: Render
- Frontend: Render
